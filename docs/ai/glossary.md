# Glossary

| Term | Meaning |
|---|---|
| Cline Skill | A reusable instruction set stored under `.cline/skills/<skill-name>/SKILL.md`. |
| `.clinerules` | Always-on project rules for Cline. |
| memory-bank | Long-term project context and progress notes. |
| Spec | Markdown documentation of expected or observed behavior. |
| Unknown | A requirement, behavior, or decision that cannot be safely determined from available evidence. |
| HTML artifact | A generated or hand-written HTML document, mockup, or deliverable reviewed against specs. |
