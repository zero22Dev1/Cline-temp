# Tech Context

## Cline Skill Structure

Each Skill lives in:

```txt
.cline/skills/<skill-name>/SKILL.md
```

Each `SKILL.md` starts with:

```md
---
name: skill-name
description: What the skill does and when to use it.
---
```

## Workspace Areas

- `.clinerules/`: always-on rules
- `.cline/skills/`: reusable workflows
- `memory-bank/`: long-term context
- `docs/ai/`: AI-facing notes, glossary, ADRs, unknowns
- `docs/specs/`: specifications
- `docs/reviews/`: reviews
